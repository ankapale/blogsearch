<h2>Skill test Questions and Answers</h2>
<p><strong>1) Is the data linearly separable?</strong></p>
<p><center><a rel="lightbox" href=https://s3-ap-south-1.amazonaws.com/av-blog-media/wp-content/uploads/2017/06/19172951/Image36.jpg><img class="alignnone wp-image-36588 size-full" src="https://s3-ap-south-1.amazonaws.com/av-blog-media/wp-content/uploads/2017/06/19172951/Image36.jpg" alt="" width="195" height="134" /></a></center><br />
A) Yes<br />
B) No</p>
<p><strong>Solution: B</strong></p>
<p>If you can draw a line or plane between the data points, it is said to be linearly separable.</p>
<p>&nbsp;</p>
<p><b>2) Which of the following are universal approximators?</b></p>
<p>A) Kernel SVM<br />
B) Neural Networks<br />
C) Boosted Decision Trees<br />
D) All of the above</p>
<p><strong>Solution: D</strong></p>
<p>All of the above methods can approximate any function.</p>
<p>&nbsp;</p>
<p><b>3) In which of the following applications can we use deep learning to solve the problem?</b></p>
<p>A) Protein structure prediction<br />
B) Prediction of chemical reactions<br />
C) Detection of exotic particles<br />
D) All of these</p>
<p><strong>Solution: D</strong></p>
<p>We can use neural network to approximate any function so it can theoretically be used to solve any problem.</p>
<p>&nbsp;</p>
<p><b>4) Which of the following statements is true when you use 1&#215;1 convolutions in a CNN?</b></p>
<p>A) It can help in dimensionality reduction<br />
B) It can be used for feature pooling<br />
C) It suffers less overfitting due to small kernel size<br />
D) All of the above</p>
<p><strong>Solution: D</strong></p>
<p>1&#215;1 convolutions are called bottleneck structure in CNN.</p>
<p>&nbsp;</p>
<p><strong><b>5) </b>Question Context:</strong></p>
<p>Statement 1: It is possible to train a network well by initializing all the weights as 0<br />
Statement 2: It is possible to train a network well by initializing biases as 0</p>
<p><b>Which of the statements given above is true?</b></p>
<p>A) Statement 1 is true while Statement 2 is false<br />
B) Statement 2 is true while statement 1 is false<br />
C) Both statements are true<br />
D) Both statements are false</p>
<p><strong>Solution: B</strong></p>
<p>Even if all the biases are zero, there is a chance that neural network may learn. On the other hand, if all the weights are zero; the neural neural network may never learn to perform the task.</p>
<p>&nbsp;</p>
<p><b>6) The number of nodes in the input layer is 10 and the hidden layer is 5. The maximum number of connections from the input layer to the hidden layer are</b></p>
<p>A) 50<br />
B) Less than 50<br />
C) More than 50<br />
D) It is an arbitrary value</p>
<p><strong>Solution: A</strong></p>
<p>Since MLP is a fully connected directed graph, the number of connections are a multiple of number of nodes in input layer and hidden layer.</p>
<p>&nbsp;</p>
<p><b>7) The input image has been converted into a matrix of size 28 X 28 and a kernel/filter of size 7 X 7 with a stride of 1. What will be the size of the convoluted matrix?</b></p>
<p>A) 22 X 22<br />
B) 21 X 21<br />
C) 28 X 28<br />
D) 7 X 7</p>
<p><strong>Solution: A</strong></p>
<p>The size of the convoluted matrix is given by C=((I-F+2P)/S)+1, where C is the size of the Convoluted matrix, I is the size of the input matrix, F the size of the filter matrix and P the padding applied to the input matrix. Here P=0, I=28, F=7 and S=1.  There the answer is 22.</p>
<p>&nbsp;</p>
<p><b>8) In a simple MLP model with 8 neurons in the input layer, 5 neurons in the hidden layer and 1 neuron in the output layer. What is the size of the weight matrices between hidden output layer and input hidden layer?</b></p>
<p>A) [1 X 5] , [5 X 8]
<p>B) [8 X 5] , [ 1 X 5]
<p>C) [8 X 5] , [5 X 1]
<p>D) [5 x 1] , [8 X 5]
<p><strong>Solution: D</strong></p>
<p>The size of weights between any layer 1 and layer 2 Is given by [nodes in layer 1 X nodes in layer 2]
<p>&nbsp;</p>
