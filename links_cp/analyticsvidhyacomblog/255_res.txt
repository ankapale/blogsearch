<h2>Skill test Questions and Answers</h2>
<p><b>1) True-False: Is Logistic regression a supervised machine learning algorithm?</b></p>
<p>A)TRUE<br />
B)FALSE</p>
<p><strong>Solution:</strong> A</p>
<p class="Standard"><span lang="EN-IN">True, Logistic regression is a supervised learning algorithm because it uses true labels for training. Supervised learning algorithm should have input variables (x) and an target variable (Y) when you train the model .</span></p>
<p>&nbsp;</p>
<p><b>2) True-False: Is Logistic regression mainly used for Regression?</b></p>
<p>A)TRUE<br />
B)FALSE</p>
<p><strong>Solution: B</strong></p>
<p class="Standard"><span lang="EN-IN">Logistic regression is a classification algorithm, don&#8217;t confuse with the name regression.</span></p>
<p>&nbsp;</p>
<p><b>3) True-False: Is it possible to design a logistic regression algorithm using a Neural Network Algorithm?</b></p>
<p>A)TRUE<br />
B)FALSE</p>
<p><strong>Solution: A</strong></p>
<p>True, Neural network is a is a <em>universal</em> approximator so it can implement linear regression algorithm.</p>
<p>&nbsp;</p>
<p><b>4) True-False: Is it possible to apply a logistic regression algorithm on a 3-class Classification problem?</b></p>
<p>A)TRUE<br />
B)FALSE</p>
<p><strong>Solution: A</strong></p>
<p class="Standard"><span lang="EN-IN">Yes, we can apply logistic regression on 3 classification problem, We can use One Vs all method for 3 class classification in logistic regression.</span></p>
<p>&nbsp;</p>
<p><b>5) Which of the following methods do we use to best fit the data in Logistic Regression?</b></p>
<p>A)Least Square Error<br />
B)Maximum Likelihood<br />
C)Jaccard distance<br />
D)Both A and B</p>
<p><strong>Solution: B</strong></p>
<p class="Standard"><span lang="EN-IN">Logistic regression uses maximum likely hood estimate for training a logistic regression.</span></p>
<p>&nbsp;</p>
<p><b>6) Which of the following evaluation metrics can not be applied in case of logistic regression output to compare with target?</b></p>
<p>A)AUC-ROC<br />
B)Accuracy<br />
C) Logloss<br />
D) Mean-Squared-Error</p>
<p><strong>Solution: D</strong></p>
<p class="Standard"><span lang="EN-IN">Since, Logistic Regression is a classification algorithm so it&#8217;s output can not be real time value so mean squared error can not use for evaluating it</span></p>
<p>&nbsp;</p>
<p><b>7) One of the very good methods to analyze the performance of Logistic Regression is AIC, which is similar to R-Squared in Linear Regression. Which of the following is true about AIC?</b></p>
<p>A) We prefer a model with minimum AIC value<br />
B) We prefer a model with maximum AIC value<br />
C)Both but depend on the situation<br />
D)None of these</p>
<p><strong>Solution: A</strong></p>
<p class="Standard"><span lang="EN-IN">We select the best model in logistic regression which can least AIC. For more information refer this source: <a href="http://www4.ncsu.edu/~shu3/Presentation/AIC.pdf"><span style="color: windowtext;text-decoration: none">http://www4.ncsu.edu/~shu3/Presentation/AIC.pdf</span></a></span></p>
<p>&nbsp;</p>
<p><b>8) [True-False] Standardisation of features is required before training a Logistic Regression.</b></p>
<p>A)TRUE<br />
B)FALSE</p>
<p><strong>Solution: B</strong></p>
<p>Standardization isn&#8217;t required for logistic regression. The main goal of standardizing features is to help convergence of the technique used for optimization.</p>
<p>&nbsp;</p>
<p><b>9) Which of the following algorithms do we use for Variable Selection?</b></p>
<p>A) LASSO<br />
B)Ridge<br />
C)Both<br />
D)None of these</p>
<p><strong>Solution: A</strong></p>
<p class="Standard"><span lang="EN-IN" style="font-family: 'Abyssinica SIL';color: #242729">In case of lasso we apply a absolute penality, after increasing the penality in lasso some of the coefficient of variables may become zero.</span></p>
<p>&nbsp;</p>

<p>For values of <em>x</em> in the range of real number from  to + Logistic function will give the output between (0,1)</p>
